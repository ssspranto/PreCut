# PreCut v1.1.0 Release Notes 🎬

This update focuses on solving a critical pain point for video editors: **codec compatibility!**

We've heard that `av01` (AV1) encoded videos were causing headaches in Adobe Premiere Pro. In this version, we've implemented a smarter download engine and a user-facing codec selector to put the control back in your hands.

## 🚀 What's New?

### 🎞️ Codec Selector
You can now choose your preferred video codec in the **Settings** page for both Clips and Proxies.
- **H.264 (Compatible)**: Default. Best for Premiere Pro, DaVinci Resolve, and older systems.
- **AV1 (Efficient)**: Best for saving storage space while maintaining high quality.
- **VP9 (Highest Quality)**: YouTube's native high-quality codec.

### ⚙️ Power User Commands
For those who like to tinker, **PreCut** now stores the full `yt-dlp` command strings in your persistent `settings.json`.
- Want to add `--cookies`?
- Need to use a specific `--proxy`?
- You can now manually edit the commands in `~/Documents/PreCut/data/settings.json` and they will be used by the app!

## 🛠️ Bug Fixes & Improvements
- **Premiere Pro Fix**: Default format filters now prioritize `avc1` (H.264) to ensure instant import into your timelines.
- **Improved Settings Migration**: Smooth transition of your existing settings to the new command-based storage engine.
- **Refined UI**: Minor spacing adjustments in the Settings page for the new dropdowns.
